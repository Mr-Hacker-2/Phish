How this works (Learning)

The Bridge: This site uses PyScript. It allows Python to talk to your browser via a JavaScript "bridge".

The Sandbox: Notice we can see your screen size and CPU count, but NOT your files, name, or IP address (directly). Browsers prevent "sandboxed" code from touching your hard drive.

Local Python: If you ran this same code as a .py file on your desktop, you could use os.listdir('/') to see every file on your computer.
